## Sección 5 - Tree

Para generar la imagen svg, uso graphviz.
El archivo fuente es "otros/genealogia.txt"
Para compilarlo, es necesario tener instalado graphviz y ejecutar 
dot -Tsvg genealogia.txt -o genealogia.svg

El archivo SVG tendrá un encabezado XML, hay que removerlo y sólo el nodo SVG copiarlo
directamente dentro del cuerpo de tree.html
